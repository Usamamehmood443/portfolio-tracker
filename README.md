# Portfolio Website

A modern, responsive portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Overview

This project serves as a professional portfolio to showcase projects, skills, and experience. Built with cutting-edge web technologies, it provides a fast, accessible, and visually appealing platform for presenting work and connecting with potential clients or employers.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Linting**: ESLint

## Project Structure

```
portfolio/
├── app/              # Next.js app router pages and layouts
├── components/       # Reusable React components
├── lib/             # Utility functions and helpers
├── public/          # Static assets (images, fonts, etc.)
└── node_modules/    # Project dependencies
```

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

The page will automatically reload when you make changes to the code.

### Building for Production

Create an optimized production build:

```bash
npm run build
```

Start the production server:

```bash
npm start
```

## Features

- Server-side rendering with Next.js App Router
- Type-safe development with TypeScript
- Responsive design with Tailwind CSS
- Optimized fonts with next/font
- ESLint for code quality

## Development Guidelines

- Place reusable components in `/components`
- Put utility functions and helpers in `/lib`
- Use TypeScript for type safety
- Follow ESLint rules for consistent code style
- Write semantic, accessible HTML
